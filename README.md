# US Emissions Dashboard (Databricks SQL)

## Overview
This project analyzes emissions across the United States using
EPA emissions data (2023). The dashboard was built using **Databricks Free
Edition** and SQL.

## Key Questions Answered
- Which states produce the highest total emissions?
- How are emissions distributed geographically?
- Which counties have the highest emissions per person?
- How concentrated are emissions among the top states?

## Tools Used
- Databricks Community Edition
- Databricks SQL
- SQL (aggregation, CTEs, null handling, data cleaning)

## Repository Contents
- `master.sql` – Full SQL logic powering the dashboard
- `dashboard.png` – Visual preview of the dashboard
